AIDE

Titre

La première ligne de votre annonce est celle que les clients sont les plus enclins à remarquer. Pensez à inclure des mots clés, de manière à ce qu'ils remarquent les titres qui correspondent à ce qu'ils recherchent. Dans notre exemple d'annonce, le mot clé "publicité" devrait attirer l'attention de ceux qui recherchent des façons de faire de la publicité en ligne.

URL à afficher

Affichée généralement en vert, l'URL à afficher précise l'adresse de votre site Web. Elle donne aux utilisateurs une idée de la page vers laquelle ils seront redirigés s'ils cliquent sur votre annonce. Vous pouvez diriger les internautes vers une partie spécifique de ce site Web à l'aide d'une URL de page de destination qu'AdWords n'affiche pas.

Description

Utilisez la description pour mettre en avant des informations précises sur vos produits ou services. Sur le mobile, où l'espace est restreint, AdWords raccourcit parfois ou supprime la ligne de description 2.
